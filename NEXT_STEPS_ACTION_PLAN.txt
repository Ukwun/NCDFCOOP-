## SUPPORT QUICK REFERENCE

### Common Issues & Fixes

**"Firebase not initialized"**
- Solution: Ensure Firebase Console project is active
- Action: Restart app, check internet connection

**"Empty product list"**
- Solution: Firestore is empty or rules are blocking
- Action: Check Firebase Console â†’ products collection

**"Payment failed"**
- Solution: Test mode vs production, invalid card
- Action: Use test card numbers from Flutterwave/Paystack docs

**"Notifications not received"**
- Solution: FCM token not saved or permissions denied
- Action: Check Android permissions in Settings

**"App crashes on startup"**
- Solution: Firebase config missing or invalid
- Action: Verify firebase_options.dart has correct project config
