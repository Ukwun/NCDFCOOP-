WHAT IF SOMETHING GOES WRONG
═════════════════════════════════════════════════════════════════════════════

Clients Report Installation Issues?
└─ Ask them to enable Unknown Sources in Settings
└─ Have them clear app cache and retry
└─ Send APK via different method (email → cloud storage)

Clients Report App Crashes?
└─ Clear cache: Settings > Apps > CoopCommerce > Clear Cache
└─ Reinstall the app
└─ Ask what they were doing when it crashed
└─ Collect error message if available

Clients Report Payment Issues?
└─ Check they used correct test card
└─ Verify phone has internet connection
└─ Try on WiFi instead of mobile data
└─ Check they selected payment method

Clients Report Data Not Saving?
└─ Verify they're logged in
└─ Check internet connection is active
└─ Have them restart app
└─ Check if account was created successfully
